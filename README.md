# Учет сайтов на хостингах

Командный проект по разработке веб-приложения для учета сайтов, их расположения на серверах, принадлежности к подразделениям и ответственным лицам.

## Описание проекта
Веб-приложение для систематизации информации о сайтах компании. Позволяет:
- Вести учет всех сайтов
- Отслеживать, на каком хостинге/сервере расположен каждый сайт
- Привязывать сайты к подразделениям
- Назначать ответственных за каждый сайт
- Быстрый поиск и фильтрация по различным параметрам

## Правила работы с репозиторием
- Запрещена прямая загрузка кода в ветку `main`
- Все изменения разрабатываются в отдельных тематических ветках
- Каждое изменение вносится через Pull Request (PR)
- Обязательное ревью от тимлида (@Laggon) перед мержем
